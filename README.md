# e-plantShopping

## 🌿 Project Overview
**e-plantShopping** is a simple e-commerce web application for browsing and purchasing plants online.  
It allows users to explore different plant categories, view product details, and simulate a shopping experience.

## 🚀 Features
- Browse a collection of plants
- View detailed product information
- Add items to cart and adjust quantity
- Responsive and user-friendly UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- React, Redux
